# Personal Resume Website

This project is a personal resume website built using HTML.  
It presents professional and academic details in a clean, simple, and organized format.

## Features
- Profile section with photo and professional summary
- Education details including degree, institution, and academic performance
- Project section with GitHub repository links and detailed descriptions
- Technical skills displayed in a table format
- Certifications and extracurricular activities
- Contact page navigation link

## Technologies Used
- HTML5

## Purpose
This project is created to practice HTML structure and layout design, and to build a basic online resume that can be hosted using GitHub Pages.
